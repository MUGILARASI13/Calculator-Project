# Calculator Project (Java)

## Overview

This is a beginner-friendly Java Console Calculator that performs basic arithmetic operations and also supports square, cube, and power functions. It stores all calculations in a history list using ArrayList from Java Collections.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Square (x²)
* Cube (x³)
* Power (aⁿ)
* Calculation history
* Clear history

## Project Structure

```
CalculatorProject/
 ├── Calculator.java
 ├── CalculationHistory.java
 └── Main.java
```

## Usage
- The program will display a menu to select operations.
- Enter numbers when prompted.
- You can view or clear the history.
- Choose 'Exit' to close the program.
